01
⚡一键安装

打开 PowerShell（按 Win+X → 选择"终端"），依次粘贴以下 3 行命令：

npm config set prefix "E:\智能整理\openclaw\npm-global"
npm config set registry https://registry.npmmirror.com
npm install -g openclaw@latest
✅ 看到版本号 2026.x.x 即安装成功。全程约 2 分钟。

03
📝填 Key · 启动

打开配置文件，把 YOUR_API_KEY 替换成你复制的 Key：

notepad C:\Users\%USERNAME%\.openclaw\openclaw.json

找到这一行，替换即可：

"apiKey": "sk-你的API Key粘贴到这里"

保存后，启动网关：

openclaw gateway
✅ 看到 "listening on ws://127.0.0.1:18789" 即启动成功！
